# **App Name**: MMCL Multimeter

## Core Features:

- Data Upload: Allow users to upload data from their multi-tester devices.
- Data Parsing: Parse uploaded data and organize it for display.
- Data Visualization: Visually represent multi-tester readings through interactive charts and graphs.
- Threshold Alert: Allow the user to specify the max value that is reasonable. If it exceeds this, add the result to the anomalies tab.
- Anomalies List: Lists all the results above the specified value.

## Style Guidelines:

- Primary color:  (#CF0A0A) to evoke trust and technical expertise.
- Background color:  (#EEEEEE), almost white, for a clean interface.
- Accent color:  (#DC5F00), slightly brighter than the primary color, to highlight interactive elements.
- Body and headline font: 'Inter', sans-serif, to maintain a neutral, legible presentation of the data.
- Use clear, professional icons to represent different types of data and tester functions.
- Single-page layout with sections for data upload, visualization, and anomaly reports.
- Subtle transitions for data loading and chart updates to enhance user experience.
